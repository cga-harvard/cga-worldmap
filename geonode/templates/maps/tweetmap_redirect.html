TweetMap

TweetMap was a WorldMap view which provided access to an interactive visualization of millions of tweets via the geospatial GPU-based platform GEOPS developed together with Todd Mostak. GEOPS was an early version of what is now OmniSci Immerse. This project also inspired development of an open source big data platform based on Lucene called the Billion Object Platform. If you have any questions about these project please contact worldmap@harvard.edu.
